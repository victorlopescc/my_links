# My Links 🌐

Projeto simples desenvolvido utilizando apenas **HTML** e **CSS**.

🔗 Acesse: https://vlopinhos.dev/

## 📌 Sobre o Projeto

O **My Links** é um site pessoal criado com o objetivo de centralizar meus principais links em um único lugar, funcionando como um **Linktree personalizado**.

Nele é possível encontrar:

- 💼 Meu currículo
- 🐙 Meu GitHub
- 🔗 Meu LinkedIn

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3

## 🎯 Objetivo

O projeto foi desenvolvido para servir como uma página simples, direta e personalizada para compartilhar minhas informações profissionais de forma prática.
